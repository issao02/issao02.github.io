Configurar GitHub Token

Para salvar notas diretamente no repositório issao02.github.io, informe um Personal Access Token com permissão de escrita no repositório.

  1. Acesse GitHub → Settings → Developer settings → Personal access tokens
  2. Crie um token (classic) com escopo repo ou fine-grained com acesso de escrita a este repositório
  3. Cole o token abaixo — fica salvo apenas no seu navegador
